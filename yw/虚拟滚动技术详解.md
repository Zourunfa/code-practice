# 虚拟滚动技术详解

## 一、什么是虚拟滚动

### 1.1 概念

虚拟滚动（Virtual Scroll）是一种优化长列表渲染性能的技术。它的核心思想是：**只渲染可视区域内的列表项，而不是渲染整个列表**。

### 1.2 为什么需要虚拟滚动

**传统列表渲染的问题：**

```typescript
// ❌ 传统方式：渲染所有数据
const MessageList = () => {
  const messages = [...]; // 假设有10000条消息
  
  return (
    <ScrollView>
      {messages.map(message => (
        <MessageItem key={message.id} data={message} />
      ))}
    </ScrollView>
  );
};
```

**问题：**
- 渲染10000个DOM节点，内存占用高（可能超过100MB）
- 首次渲染时间长（可能超过3秒）
- 滚动卡顿，帧率低（可能低于30fps）
- 页面响应慢，用户体验差

**虚拟滚动的优势：**

```typescript
// ✅ 虚拟滚动：只渲染可见部分
const VirtualMessageList = () => {
  const messages = [...]; // 10000条消息
  const visibleMessages = getVisibleMessages(); // 只有20条
  
  return (
    <ScrollView>
      {visibleMessages.map(message => (
        <MessageItem key={message.id} data={message} />
      ))}
    </ScrollView>
  );
};
```

**优势：**
- 只渲染20个DOM节点，内存占用低（约2MB）
- 首次渲染快（约100ms）
- 滚动流畅，帧率高（60fps）
- 用户体验好

### 1.3 适用场景

**适合使用虚拟滚动的场景：**
- 长列表（数据量 > 100条）
- 聊天消息列表
- 商品列表
- 新闻列表
- 评论列表
- 搜索结果列表

**不适合使用虚拟滚动的场景：**
- 短列表（数据量 < 50条）
- 列表项高度差异很大且不可预测
- 需要频繁操作DOM的场景
- 列表项包含复杂交互的场景



## 二、虚拟滚动原理

### 2.1 核心原理

虚拟滚动的核心原理可以用一张图来说明：

```
┌─────────────────────────────────────────┐
│  容器（Container）                        │
│  ┌───────────────────────────────────┐  │
│  │  占位元素（Placeholder - 上）      │  │ ← 不渲染，只占位
│  │  高度 = startIndex * itemHeight   │  │
│  ├───────────────────────────────────┤  │
│  │  ┌─────────────────────────────┐ │  │
│  │  │  可见区域（Viewport）        │ │  │
│  │  │  ┌───────────────────────┐  │ │  │
│  │  │  │  Item 10              │  │ │  │ ← 渲染
│  │  │  ├───────────────────────┤  │ │  │
│  │  │  │  Item 11              │  │ │  │ ← 渲染
│  │  │  ├───────────────────────┤  │ │  │
│  │  │  │  Item 12              │  │ │  │ ← 渲染
│  │  │  └───────────────────────┘  │ │  │
│  │  └─────────────────────────────┘ │  │
│  ├───────────────────────────────────┤  │
│  │  占位元素（Placeholder - 下）      │  │ ← 不渲染，只占位
│  │  高度 = (total - endIndex) * h   │  │
│  └───────────────────────────────────┘  │
└─────────────────────────────────────────┘
```

**关键要素：**

1. **容器高度**：固定高度，可滚动
2. **总高度**：`totalHeight = 数据总数 × 单项高度`
3. **可视区域**：用户能看到的区域
4. **渲染区域**：实际渲染的区域（可视区域 + 缓冲区）
5. **占位元素**：上下两个占位元素，撑起总高度

### 2.2 计算公式

```typescript
// 1. 计算可视区域能显示的数量
const visibleCount = Math.ceil(containerHeight / itemHeight);

// 2. 计算开始索引
const startIndex = Math.floor(scrollTop / itemHeight);

// 3. 计算结束索引
const endIndex = startIndex + visibleCount;

// 4. 计算上方占位高度
const offsetTop = startIndex * itemHeight;

// 5. 计算下方占位高度
const offsetBottom = (totalCount - endIndex) * itemHeight;

// 6. 获取可见数据
const visibleData = allData.slice(startIndex, endIndex);
```

### 2.3 工作流程

```
用户滚动
    ↓
监听滚动事件
    ↓
获取 scrollTop
    ↓
计算 startIndex 和 endIndex
    ↓
从数据源中切片获取可见数据
    ↓
更新占位元素高度
    ↓
渲染可见列表项
    ↓
等待下次滚动
```



## 三、基础实现

### 3.1 固定高度的虚拟滚动

这是最简单的虚拟滚动实现，适用于所有列表项高度相同的场景。

#### 3.1.1 React实现

```typescript
import React, { useState, useRef, useCallback } from 'react';

interface VirtualListProps {
  data: any[];              // 数据源
  itemHeight: number;       // 每项高度
  containerHeight: number;  // 容器高度
  renderItem: (item: any, index: number) => React.ReactNode;
}

const VirtualList: React.FC<VirtualListProps> = ({
  data,
  itemHeight,
  containerHeight,
  renderItem,
}) => {
  const [scrollTop, setScrollTop] = useState(0);
  const containerRef = useRef<HTMLDivElement>(null);
  
  // 计算可见数量
  const visibleCount = Math.ceil(containerHeight / itemHeight);
  
  // 计算开始和结束索引（添加缓冲区）
  const startIndex = Math.max(0, Math.floor(scrollTop / itemHeight) - 2);
  const endIndex = Math.min(data.length, startIndex + visibleCount + 4);
  
  // 计算占位高度
  const offsetTop = startIndex * itemHeight;
  const offsetBottom = (data.length - endIndex) * itemHeight;
  
  // 获取可见数据
  const visibleData = data.slice(startIndex, endIndex);
  
  // 滚动事件处理
  const handleScroll = useCallback((e: React.UIEvent<HTMLDivElement>) => {
    const scrollTop = e.currentTarget.scrollTop;
    setScrollTop(scrollTop);
  }, []);
  
  return (
    <div
      ref={containerRef}
      style={{
        height: containerHeight,
        overflow: 'auto',
        position: 'relative',
      }}
      onScroll={handleScroll}
    >
      {/* 上方占位 */}
      <div style={{ height: offsetTop }} />
      
      {/* 可见列表项 */}
      {visibleData.map((item, index) => (
        <div
          key={startIndex + index}
          style={{ height: itemHeight }}
        >
          {renderItem(item, startIndex + index)}
        </div>
      ))}
      
      {/* 下方占位 */}
      <div style={{ height: offsetBottom }} />
    </div>
  );
};

export default VirtualList;
```

**使用示例：**

```typescript
const App = () => {
  // 生成10000条数据
  const data = Array.from({ length: 10000 }, (_, i) => ({
    id: i,
    text: `消息 ${i}`,
  }));
  
  return (
    <VirtualList
      data={data}
      itemHeight={50}
      containerHeight={600}
      renderItem={(item) => (
        <div style={{ padding: '10px', borderBottom: '1px solid #eee' }}>
          {item.text}
        </div>
      )}
    />
  );
};
```

### 3.2 动态高度的虚拟滚动

适用于列表项高度不固定的场景，需要预估高度或实际测量。

```typescript
interface VirtualListDynamicProps {
  data: any[];
  estimatedItemHeight: number;  // 预估高度
  containerHeight: number;
  renderItem: (item: any, index: number) => React.ReactNode;
}

const VirtualListDynamic: React.FC<VirtualListDynamicProps> = ({
  data,
  estimatedItemHeight,
  containerHeight,
  renderItem,
}) => {
  const [scrollTop, setScrollTop] = useState(0);
  const [itemHeights, setItemHeights] = useState<Map<number, number>>(new Map());
  const itemRefs = useRef<Map<number, HTMLDivElement>>(new Map());
  
  // 获取或预估项的高度
  const getItemHeight = (index: number): number => {
    return itemHeights.get(index) || estimatedItemHeight;
  };
  
  // 计算项的偏移位置
  const getItemOffset = (index: number): number => {
    let offset = 0;
    for (let i = 0; i < index; i++) {
      offset += getItemHeight(i);
    }
    return offset;
  };
  
  // 计算总高度
  const totalHeight = useMemo(() => {
    let height = 0;
    for (let i = 0; i < data.length; i++) {
      height += getItemHeight(i);
    }
    return height;
  }, [data.length, itemHeights]);
  
  // 查找可见范围
  const getVisibleRange = useCallback(() => {
    let startIndex = 0;
    let endIndex = 0;
    let currentOffset = 0;
    
    // 找到开始索引
    for (let i = 0; i < data.length; i++) {
      const itemHeight = getItemHeight(i);
      if (currentOffset + itemHeight > scrollTop) {
        startIndex = Math.max(0, i - 2); // 添加缓冲
        break;
      }
      currentOffset += itemHeight;
    }
    
    // 找到结束索引
    currentOffset = getItemOffset(startIndex);
    for (let i = startIndex; i < data.length; i++) {
      const itemHeight = getItemHeight(i);
      currentOffset += itemHeight;
      if (currentOffset > scrollTop + containerHeight) {
        endIndex = Math.min(data.length, i + 2); // 添加缓冲
        break;
      }
    }
    
    if (endIndex === 0) {
      endIndex = data.length;
    }
    
    return { startIndex, endIndex };
  }, [scrollTop, data.length, containerHeight, itemHeights]);
  
  const { startIndex, endIndex } = getVisibleRange();
  const visibleData = data.slice(startIndex, endIndex);
  const offsetTop = getItemOffset(startIndex);
  
  // 测量项的实际高度
  const measureItem = useCallback((index: number, element: HTMLDivElement | null) => {
    if (element) {
      const height = element.getBoundingClientRect().height;
      if (height !== getItemHeight(index)) {
        setItemHeights(prev => {
          const newMap = new Map(prev);
          newMap.set(index, height);
          return newMap;
        });
      }
      itemRefs.current.set(index, element);
    }
  }, []);
  
  const handleScroll = useCallback(
    throttle((e: React.UIEvent<HTMLDivElement>) => {
      setScrollTop(e.currentTarget.scrollTop);
    }, 16),
    []
  );
  
  return (
    <div
      style={{
        height: containerHeight,
        overflow: 'auto',
        position: 'relative',
      }}
      onScroll={handleScroll}
    >
      <div style={{ height: totalHeight, position: 'relative' }}>
        {visibleData.map((item, index) => {
          const actualIndex = startIndex + index;
          const offset = getItemOffset(actualIndex);
          
          return (
            <div
              key={actualIndex}
              ref={(el) => measureItem(actualIndex, el)}
              style={{
                position: 'absolute',
                top: offset,
                left: 0,
                right: 0,
              }}
            >
              {renderItem(item, actualIndex)}
            </div>
          );
        })}
      </div>
    </div>
  );
};
```



## 四、Taro中的虚拟滚动实现

### 4.1 使用Taro的VirtualList组件

Taro 3.x 提供了内置的 `VirtualList` 组件，支持小程序和H5。

#### 4.1.1 基础使用

```typescript
import { VirtualList } from '@tarojs/components';
import { View, Text } from '@tarojs/components';

const MessageList = () => {
  // 数据源
  const messages = Array.from({ length: 10000 }, (_, i) => ({
    id: i,
    text: `消息 ${i}`,
    time: new Date().toLocaleTimeString(),

      height={600}              // 容器高度
      width="100%"              // 容器宽度
      itemData={messages}       // 数据源
      itemCount={messages.length} // 数据数量
      itemSize={80}             // 每项高度
      item={renderItem}         // 渲染函数
    />
  );
};

export default MessageList;
```

#### 4.1.2 动态高度实现

```typescript
import { VirtualList } from '@tarojs/components';
import { useState, useCallback } from 'react';

const DynamicMessageList = () => {
  const [itemHeights, setItemHeights] = useState<Map<number, number>>(new Map());
  
  const messages = [...]; // 数据源

                return newMap;
              });
            }
          });
        }}
        id={`item-${index}`}
      >
        <Text>{item.text}</Text>
      </View>
    );
  }, []);
  
  return (
    <VirtualList
      height={600}
      width="100%"
      itemData={messages}
      itemCount={messages.length}
      itemSize={getItemSize}  // 传入函数
      item={renderItem}
    />
  );
```

### 4.2 自定义Taro虚拟滚动

如果需要更灵活的控制，可以自己实现虚拟滚动。

#### 4.2.1 使用ScrollView实现

```typescript
import { ScrollView, View } from '@tarojs/components';
import { useState, useMemo } from 'react';
import Taro from '@tarojs/taro';

interface VirtualScrollViewProps {
  data: any[];
  itemHeight: number;
  renderItem: (item: any, index: number) => React.ReactNode;
}

const VirtualScrollView: React.FC<VirtualScrollViewProps> = ({
  data,
  itemHeight,
  renderItem,
}) => {
  const [scrollTop, setScrollTop] = useState(0);
  
  // 获取屏幕高度
  const screenHeight = useMemo(() => {
    const { windowHeight } = Taro.getSystemInfoSync();
    return windowHeight;
  }, []);
  
  // 计算可见范围
  const { startIndex, endIndex, offsetTop, offsetBottom, visibleData } = useMemo(() => {
    const visibleCount = Math.ceil(screenHeight / itemHeight);
    const startIndex = Math.max(0, Math.floor(scrollTop / itemHeight) - 3);
    const endIndex = Math.min(data.length, startIndex + visibleCount + 6);
    const offsetTop = startIndex * itemHeight;
    const offsetBottom = (data.length - endIndex) * itemHeight;
    const visibleData = data.slice(startIndex, endIndex);
    
    return { startIndex, endIndex, offsetTop, offsetBottom, visibleData };
  }, [scrollTop, data, itemHeight, screenHeight]);
  
  // 滚动事件处理
  const handleScroll = (e) => {
    const { scrollTop } = e.detail;
    setScrollTop(scrollTop);
  };
  
  return (
    <ScrollView
      scrollY
      style={{ height: '100vh' }}
      onScroll={handleScroll}
      scrollTop={scrollTop}
    >
      {/* 上方占位 */}
      <View style={{ height: `${offsetTop}px` }} />
      
      {/* 可见列表项 */}
      {visibleData.map((item, index) => (
        <View
          key={startIndex + index}
          style={{ height: `${itemHeight}px` }}
        >
          {renderItem(item, startIndex + index)}
        </View>
      ))}
      
      {/* 下方占位 */}
      <View style={{ height: `${offsetBottom}px` }} />
    </ScrollView>
  );
};

export default VirtualScrollView;
```

#### 4.2.2 使用MobX优化

```typescript
import { observable, computed, action, makeObservable } from 'mobx';
import { observer } from 'mobx-react';
import Taro from '@tarojs/taro';

class VirtualListStore {
  @observable scrollTop = 0;
  @observable data: any[] = [];
  
  itemHeight = 80;
  screenHeight = 0;
  bufferSize = 3; // 缓冲区大小
  
  constructor() {
    makeObservable(this);
    const { windowHeight } = Taro.getSystemInfoSync();
    this.screenHeight = windowHeight;
  }
  
  @action setScrollTop(scrollTop: number) {
    this.scrollTop = scrollTop;
  }
  
  @action setData(data: any[]) {
    this.data = data;
  }
  
  @computed get visibleCount() {
    return Math.ceil(this.screenHeight / this.itemHeight);
  }
  
  @computed get startIndex() {
    return Math.max(0, Math.floor(this.scrollTop / this.itemHeight) - this.bufferSize);
  }
  
  @computed get endIndex() {
    return Math.min(
      this.data.length,
      this.startIndex + this.visibleCount + this.bufferSize * 2
    );
  }
  
  @computed get offsetTop() {
    return this.startIndex * this.itemHeight;
  }
  
  @computed get offsetBottom() {
    return (this.data.length - this.endIndex) * this.itemHeight;
  }
  
  @computed get visibleData() {
    return this.data.slice(this.startIndex, this.endIndex);
  }
}

// 使用
const VirtualList = observer(() => {
  const [store] = useState(() => new VirtualListStore());
  
  useEffect(() => {
    // 加载数据
    const data = Array.from({ length: 10000 }, (_, i) => ({
      id: i,
      text: `消息 ${i}`,
    }));
    store.setData(data);
  }, []);
  
  const handleScroll = (e) => {
    store.setScrollTop(e.detail.scrollTop);
  };
  
  return (
    <ScrollView
      scrollY
      style={{ height: '100vh' }}
      onScroll={handleScroll}
    >
      <View style={{ height: `${store.offsetTop}px` }} />
      
      {store.visibleData.map((item, index) => (
        <View
          key={store.startIndex + index}
          style={{ height: `${store.itemHeight}px` }}
        >
          <Text>{item.text}</Text>
        </View>
      ))}
      
      <View style={{ height: `${store.offsetBottom}px` }} />
    </ScrollView>
  );
});
```

### 4.3 聊天消息列表实现

针对IM聊天场景的虚拟滚动实现。

```typescript
import { ScrollView, View, Text } from '@tarojs/components';
import { observer } from 'mobx-react';
import { computed } from 'mobx';

@observer
class ChatMessageList extends React.Component {
  @observable scrollTop = 0;
  @observable messages = [];
  
  itemHeight = 100; // 预估高度
  screenHeight = Taro.getSystemInfoSync().windowHeight;
  
  @computed get visibleMessages() {
    const visibleCount = Math.ceil(this.screenHeight / this.itemHeight);
    const startIndex = Math.max(0, Math.floor(this.scrollTop / this.itemHeight) - 5);
    const endIndex = Math.min(
      this.messages.length,
      startIndex + visibleCount + 10
    );
    
    return {
      startIndex,
      endIndex,
      offsetTop: startIndex * this.itemHeight,
      offsetBottom: (this.messages.length - endIndex) * this.itemHeight,
      data: this.messages.slice(startIndex, endIndex),
    };
  }
  
  handleScroll = (e) => {
    this.scrollTop = e.detail.scrollTop;
  };
  
  // 滚动到底部
  scrollToBottom = () => {
    const totalHeight = this.messages.length * this.itemHeight;
    this.scrollTop = totalHeight;
  };
  
  // 添加新消息
  addMessage = (message) => {
    this.messages.push(message);
    // 延迟滚动到底部
    setTimeout(() => {
      this.scrollToBottom();
    }, 100);
  };
  
  render() {
    const { startIndex, offsetTop, offsetBottom, data } = this.visibleMessages;
    
    return (
      <ScrollView
        scrollY
        style={{ height: '100vh' }}
        scrollTop={this.scrollTop}
        onScroll={this.handleScroll}
      >
        <View style={{ height: `${offsetTop}px` }} />
        
        {data.map((message, index) => (
          <View
            key={startIndex + index}
            className="message-item"
            style={{ minHeight: `${this.itemHeight}px` }}
          >
            <Text>{message.content}</Text>
            <Text className="time">{message.time}</Text>
          </View>
        ))}
        
        <View style={{ height: `${offsetBottom}px` }} />
      </ScrollView>
    );
  }
}

export default ChatMessageList;
```

### 5.1 滚动事件优化

#### 5.1.1 使用节流（Throttle）

```typescript
import { throttle } from 'lodash';

const handleScroll = useCallback(
  throttle((e) => {
    const scrollTop = e.detail.scrollTop;
    setScrollTop(scrollTop);
```

**场景描述：**

**技术方案：**

```typescript
// InquiryDetailStore.ts
class InquiryDetailStore extends BaseStore {
  @observable messages: IChatMessage[] = [];
  @observable scrollTop = 0;
  
  // 消息项预估高度
  private estimateMessageHeight = (message: IChatMessage) => {
    const baseHeight = 60;
    
    // 根据消息类型估算
    switch (message.type) {
      case EIMMessageType.TEXT:
        return baseHeight + Math.ceil(message.content.length / 20) * 20;
      case EIMMessageType.IMAGE:
        return 200;
      case EIMMessageType.CUSTOM:
        return 120;
      default:
        return baseHeight;
    }
  };
  
  // 计算可见消息
  @computed get visibleMessages() {
    const screenHeight = Taro.getSystemInfoSync().windowHeight;
    const bufferSize = 5;
    
    let currentHeight = 0;
    let startIndex = 0;
    
    // 从当前滚动位置向上查找起始索引
    for (let i = 0; i < this.messages.length; i++) {
      const itemHeight = this.estimateMessageHeight(this.messages[i]);
      if (currentHeight + itemHeight > this.scrollTop) {
        startIndex = Math.max(0, i - bufferSize);
        break;
      }
      currentHeight += itemHeight;
    }
    
    // 从起始索引向下查找结束索引
    currentHeight = 0;
    let endIndex = startIndex;
    for (let i = startIndex; i < this.messages.length; i++) {
      const itemHeight = this.estimateMessageHeight(this.messages[i]);
      currentHeight += itemHeight;
      if (currentHeight > screenHeight + this.scrollTop) {
        endIndex = Math.min(this.messages.length, i + bufferSize);
        break;
      }
    }
    
    return {
      startIndex,
      endIndex,
      data: this.messages.slice(startIndex, endIndex),
    };
  }
  
  // 添加新消息并滚动到底部
  @action addMessage = (message: IChatMessage) => {
    this.messages.push(message);
    
    // 延迟滚动到底部
    setTimeout(() => {
      this.scrollToBottom();
    }, 100);
  };
  
  // 滚动到底部
  @action scrollToBottom = () => {
    let totalHeight = 0;
    this.messages.forEach(msg => {
      totalHeight += this.estimateMessageHeight(msg);
    });
    this.scrollTop = totalHeight;
  };
}
```

**技术方案：**

```typescript
const CommentList = () => {
  const [comments, setComments] = useState([]);
  const [scrollTop, setScrollTop] = useState(0);
  const [itemHeights, setItemHeights] = useState(new Map());
  
  // 预估高度
  const estimateHeight = (comment) => {
    const baseHeight = 80;
    const textHeight = Math.ceil(comment.content.length / 30) * 20;
    const imageHeight = comment.images?.length ? 100 : 0;
    return baseHeight + textHeight + imageHeight;
  };
  
  // 获取实际或预估高度
  const getItemHeight = (index) => {
    return itemHeights.get(index) || estimateHeight(comments[index]);
  };
  
  // 计算偏移量
  const getItemOffset = (index) => {
    let offset = 0;
    for (let i = 0; i < index; i++) {
      offset += getItemHeight(i);
    }
    return offset;
  };
  
  // 查找可见范围
  const getVisibleRange = () => {
    const screenHeight = Taro.getSystemInfoSync().windowHeight;

    let endIndex = 0;

    for (let i = 0; i < comments.length; i++) {
      const height = getItemHeight(i);
      if (currentOffset + height > scrollTop) {
        startIndex = Math.max(0, i - 2);
        break;
      }
      currentOffset += height;
    }
    
    // 查找结束索引
    currentOffset = getItemOffset(startIndex);
    for (let i = startIndex; i < comments.length; i++) {
      const height = getItemHeight(i);
      currentOffset += height;
      if (currentOffset > scrollTop + screenHeight) {
        endIndex = Math.min(comments.length, i + 2);
        break;
      }
    }
    
    return { startIndex, endIndex: endIndex || comments.length };
  };
  
  const { startIndex, endIndex } = getVisibleRange();
  const visibleData = comments.slice(startIndex, endIndex);
  const offsetTop = getItemOffset(startIndex);
  
  // 测量实际高度
  const measureHeight = (index, element) => {
    if (element) {
      const height = element.getBoundingClientRect().height;
      if (height !== getItemHeight(index)) {
        setItemHeights(prev => new Map(prev).set(index, height));
      }
    }
  };
  
  return (
    <ScrollView
      scrollY
      style={{ height: '100vh' }}
      onScroll={(e) => setScrollTop(e.detail.scrollTop)}
    >
      <View style={{ position: 'relative' }}>

          return (
            <View
              key={comment.id}
              ref={(el) => measureHeight(actualIndex, el)}
              style={{
                position: 'absolute',
                top: `${offset}px`,
                left: 0,
                right: 0,
              }}
```

#### 固定高度虚拟列表（生产级）

```typescript
import React, { useState, useRef, useCallback, useMemo, useEffect } from 'react';
import { throttle } from 'lodash';

interface VirtualListProps<T> {
  data: T[];
  itemHeight: number;
  containerHeight: number;
  renderItem: (item: T, index: number) => React.ReactNode;
  onLoadMore?: () => void;
  bufferSize?: number;
}

function VirtualList<T extends { id: string | number }>({
  data,
  itemHeight,
  containerHeight,
  renderItem,
  onLoadMore,
  bufferSize = 3,
}: VirtualListProps<T>) {
  const [scrollTop, setScrollTop] = useState(0);
  const containerRef = useRef<HTMLDivElement>(null);
  const isLoadingMore = useRef(false);
  
  // 计算可见范围
  const { startIndex, endIndex, offsetTop, offsetBottom, visibleData } = useMemo(() => {
    const visibleCount = Math.ceil(containerHeight / itemHeight);
    const startIndex = Math.max(0, Math.floor(scrollTop / itemHeight) - bufferSize);
    const endIndex = Math.min(data.length, startIndex + visibleCount + bufferSize * 2);
    const offsetTop = startIndex * itemHeight;
    const offsetBottom = (data.length - endIndex) * itemHeight;
    const visibleData = data.slice(startIndex, endIndex);
    
    return { startIndex, endIndex, offsetTop, offsetBottom, visibleData };
  }, [scrollTop, data, itemHeight, containerHeight, bufferSize]);
  
  // 滚动事件处理（节流）
  const handleScroll = useCallback(
    throttle((e: React.UIEvent<HTMLDivElement>) => {
      const scrollTop = e.currentTarget.scrollTop;
      setScrollTop(scrollTop);
      
      // 触底加载更多
      const scrollHeight = e.currentTarget.scrollHeight;
      const clientHeight = e.currentTarget.clientHeight;
      
      if (
        onLoadMore &&
        !isLoadingMore.current &&
        scrollTop + clientHeight >= scrollHeight - 100
      ) {
        isLoadingMore.current = true;
        onLoadMore();
        setTimeout(() => {
          isLoadingMore.current = false;
        }, 1000);
      }
    }, 16),
    [onLoadMore]
  );
  
  // 清理
  useEffect(() => {
    return () => {
      handleScroll.cancel();
    };
  }, [handleScroll]);
  
  return (
    <div
      ref={containerRef}
      style={{
        height: containerHeight,
        overflow: 'auto',
        position: 'relative',
      }}
      onScroll={handleScroll}
    >
      {/* 上方占位 */}
      <div style={{ height: offsetTop }} />
      
      {/* 可见列表项 */}
      {visibleData.map((item, index) => (
        <div
          key={item.id}
          style={{ height: itemHeight }}
        >
          {renderItem(item, startIndex + index)}
        </div>
      ))}
      
      {/* 下方占位 */}
      <div style={{ height: offsetBottom }} />
    </div>
  );
}

export default VirtualList;
```

#### 动态高度虚拟列表（生产级）

```typescript
import React, { useState, useRef, useCallback, useMemo, useEffect } from 'react';
import { throttle } from 'lodash';

interface DynamicVirtualListProps<T> {
  data: T[];
  estimatedItemHeight: number;
  containerHeight: number;
  renderItem: (item: T, index: number) => React.ReactNode;
  bufferSize?: number;
}

function DynamicVirtualList<T extends { id: string | number }>({
  data,
  estimatedItemHeight,
  containerHeight,
  renderItem,
  bufferSize = 3,
}: DynamicVirtualListProps<T>) {
  const [scrollTop, setScrollTop] = useState(0);
  const [itemHeights, setItemHeights] = useState<Map<number, number>>(new Map());
  const containerRef = useRef<HTMLDivElement>(null);
  const itemRefs = useRef<Map<number, HTMLDivElement>>(new Map());
  
  // 获取项的高度
  const getItemHeight = useCallback(
    (index: number): number => {
      return itemHeights.get(index) || estimatedItemHeight;
    },
    [itemHeights, estimatedItemHeight]
  );
  
  // 计算项的偏移位置
  const getItemOffset = useCallback(
    (index: number): number => {
      let offset = 0;
      for (let i = 0; i < index; i++) {
        offset += getItemHeight(i);
      }
      return offset;
    },
    [getItemHeight]
  );
  
  // 计算总高度
  const totalHeight = useMemo(() => {
    let height = 0;
    for (let i = 0; i < data.length; i++) {
      height += getItemHeight(i);
    }
    return height;
  }, [data.length, getItemHeight]);
  
  // 查找可见范围
  const getVisibleRange = useCallback(() => {
    let startIndex = 0;
    let endIndex = 0;
    let currentOffset = 0;
    
    // 找到开始索引
    for (let i = 0; i < data.length; i++) {
      const itemHeight = getItemHeight(i);
      if (currentOffset + itemHeight > scrollTop) {
        startIndex = Math.max(0, i - bufferSize);
        break;
      }
      currentOffset += itemHeight;
    }
    
    // 找到结束索引
    currentOffset = getItemOffset(startIndex);
    for (let i = startIndex; i < data.length; i++) {
      const itemHeight = getItemHeight(i);
      currentOffset += itemHeight;
      if (currentOffset > scrollTop + containerHeight) {
        endIndex = Math.min(data.length, i + bufferSize);
        break;
      }
    }
    
    if (endIndex === 0) {
      endIndex = data.length;
    }
    
    return { startIndex, endIndex };
  }, [scrollTop, data.length, containerHeight, getItemHeight, getItemOffset, bufferSize]);
  
  const { startIndex, endIndex } = getVisibleRange();
  const visibleData = data.slice(startIndex, endIndex);
  
  // 测量项的实际高度
  const measureItem = useCallback(
    (index: number, element: HTMLDivElement | null) => {
      if (element) {
        const height = element.getBoundingClientRect().height;
        const currentHeight = getItemHeight(index);
        
        if (Math.abs(height - currentHeight) > 1) {
          setItemHeights((prev) => {
            const newMap = new Map(prev);
            newMap.set(index, height);
            return newMap;
          });
        }
        
        itemRefs.current.set(index, element);
      }
    },
    [getItemHeight]
  );
  
  // 滚动事件处理
  const handleScroll = useCallback(
    throttle((e: React.UIEvent<HTMLDivElement>) => {
      setScrollTop(e.currentTarget.scrollTop);
    }, 16),
    []
  );
  
  // 清理
  useEffect(() => {
    return () => {
      handleScroll.cancel();
    };
  }, [handleScroll]);
  
  return (
    <div
      ref={containerRef}
      style={{
        height: containerHeight,
        overflow: 'auto',
        position: 'relative',
      }}
      onScroll={handleScroll}
    >
      <div style={{ height: totalHeight, position: 'relative' }}>
        {visibleData.map((item, index) => {
          const actualIndex = startIndex + index;
          const offset = getItemOffset(actualIndex);
          
          return (
            <div
              key={item.id}
              ref={(el) => measureItem(actualIndex, el)}
              style={{
                position: 'absolute',
                top: offset,
                left: 0,
                right: 0,
              }}
            >
              {renderItem(item, actualIndex)}
            </div>
          );
        })}
      </div>
    </div>
  );
}

export default DynamicVirtualList;
```

### 10.5 总结

虚拟滚动是优化长列表性能的重要技术，通过只渲染可见区域的内容，可以显著提升应用性能和用户体验。

**核心要点：**

1. **原理简单**：只渲染可见区域，使用占位元素撑起总高度
2. **性能提升明显**：可降低90%以上的渲染时间和内存占用
3. **实现方式多样**：固定高度、动态高度、横向滚动、双向滚动
4. **需要权衡**：增加了实现复杂度，需要处理更多边界情况
5. **适用场景广泛**：聊天列表、商品列表、评论列表、表格等

**实施建议：**

- 数据量 < 100：不需要虚拟滚动
- 数据量 100-500：考虑使用虚拟滚动
- 数据量 > 500：强烈建议使用虚拟滚动

**技术选型：**

- 简单场景：使用现成的库（react-window、@tanstack/react-virtual）
- 复杂场景：自己实现，针对性优化
- 小程序：使用Taro VirtualList或自己实现

虚拟滚动不是银弹，需要根据实际场景选择合适的方案。在实施过程中，要注重性能监控和用户体验，持续优化和改进。
